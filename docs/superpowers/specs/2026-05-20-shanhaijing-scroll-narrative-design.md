# 《山海经》卷轴叙事流 - 设计文档

> **目标**：复刻永乐大典数字化网站（识典古籍）的皇家卷轴叙事风格，为《山海经》主题打造沉浸式数字博览体验
> **风格基调**：皇家朱红 + 金色 + 象牙白，富贵典雅，典籍厚重感
> **技术栈**：React 19 + TypeScript + Vite + Tailwind CSS + GSAP + WebGL
> **设计日期**：2026-05-20

---

### 1.2 字体体系

| 用途 | 字体栈 | 说明 |
|------|--------|------|
| 大标题/卷首 | `'Noto Serif SC', 'Source Han Serif SC', 'SimSun', serif` | 宋体/思源宋体，庄重典雅 |
| 正文 | `'Noto Sans SC', 'PingFang SC', 'Microsoft YaHei', sans-serif` | 黑体，清晰易读 |

- **卷轴边框**：`border: 2px solid var(--gold-primary)` 模拟卷轴边缘
- **卷轴阴影**：`box-shadow: 0 0 40px rgba(139, 26, 26, 0.1)`

## 二、页面结构：六章卷轴

### 整体导航结构

页面顶部固定一个**卷轴式章节导航栏**，随着滚动高亮当前章节：

```
┌─────────────────────────────────────────────────────────┐
│  山海经数字博览    鸿蒙 · 山川 · 异兽 · 邦国 · 草木 · 遗典  │
└─────────────────────────────────────────────────────────┘
```

每个章节名使用书法字体，当前章节金色高亮，其他章节墨色。

### 2.1 第一章：鸿蒙（全屏卷轴开场）

**视觉定位**：永乐大典网站 "初见" 章节的对标

**布局**：
```
┌────────────────────────────────────────────┐
│                                            │
│         [卷轴缓缓展开动画]                  │
│                                            │
│              山 海 经                       │
│         ─────────────────                  │
│        上古奇书 · 志怪地理之祖              │
│                                            │
│    [卷轴装饰纹样 - 云雷纹/回纹]              │
│                                            │
│         向下展开卷轴 ↓                      │
│                                            │
└────────────────────────────────────────────┘
```

**设计细节**：
- 背景：象牙白 `var(--ivory)`，带有极淡的宣纸纹理（CSS `background-image` 或 subtle pattern）
- 中央："山海经" 三个大字，使用宋体，`font-size: clamp(4rem, 12vw, 10rem)`，墨色
- 副标题：金色细线分隔 + 小字说明
- 底部：卷轴展开提示，金色箭头动画（`@keyframes scroll-down`）
- 卷轴边框：左右两侧模拟卷轴木轴（`border-left/right: 8px solid var(--gold-muted)`）
- **入场动画**：页面加载时，卷轴从中间向两侧展开（`scaleX: 0 → 1`），文字淡入

**WebGL 着色器调整**：
- 将现有 `VoidShader.tsx` 的暗色宇宙风格改为**水墨山水风格**
- 色调改为：淡墨山水 + 朱红点缀（远山轮廓）
- 或者：完全移除 WebGL，改用纯 CSS 卷轴展开动画（更贴合古典风格）

**决策**：移除 WebGL 着色器，改用纯 CSS 卷轴动画。永乐大典网站没有复杂的 WebGL 背景，而是依靠纯粹的排版和动画营造氛围。

## 三、交互动画规范

### 3.1 卷轴展开动画（鸿蒙章节）

```css
@keyframes scroll-unroll {
  0% {
    transform: scaleX(0);
    opacity: 0;
  }
  50% {
    transform: scaleX(1);
    opacity: 0.5;
  }
  100% {
    transform: scaleX(1);
    opacity: 1;
  }
}

.scroll-container {
  animation: scroll-unroll 1.5s ease-out forwards;
  transform-origin: center;
}
```

### 3.2 章节导航高亮

- 滚动到对应章节时，导航栏中的章节名变为金色
- 过渡：`color 0.3s ease`
- 当前章节下方显示金色细线（`border-bottom: 2px solid var(--gold-primary)`）

### 3.3 数字计数动画（山川章节）

```typescript
// GSAP ScrollTrigger 驱动
gsap.to(counter, {
  innerText: 550,
  duration: 2,
  snap: { innerText: 1 },
  scrollTrigger: {
    trigger: counter,
    start: 'top 80%',
    once: true,
  },
});
```

### 3.4 神兽列表悬停

- 文字颜色：墨色 → 朱红色（`color 0.3s ease`）
- 右侧图片：从 `opacity: 0, scale: 0.8` 到 `opacity: 1, scale: 1`
- 图片边框：金色卷轴式边框（`border: 2px solid var(--gold-primary)`）

### 3.5 页面过渡

- 点击神兽进入详情页：卷轴从点击位置向两侧展开的过渡动画
- 返回：卷轴向中间收起的过渡动画

## 七、性能考量

1. **字体加载**：使用 `font-display: swap`，避免 FOIT
2. **动画性能**：使用 `transform` 和 `opacity`，避免触发重排
3. **图片懒加载**：神兽图片使用 IntersectionObserver
4. **卷轴纹理**：使用 CSS 渐变模拟，避免大图片资源
